# 10 Freelance Proposal Templates

## Template 1: Web Development
**Subject:** Website build — quote & timeline
**Body:** Hi [Name], I can build your [type] website. Here's what I propose: [scope]. Timeline: [time]. Investment: [price]. Portfolio: [link]. Let me know if you want to discuss!

## Template 2: Excel/Automation
**Subject:** Excel automation solution
**Body:** Hi, I can automate your spreadsheet in under a week. I'll build formulas, macros, and a clean dashboard. Price: $[amount]. Delivered in [time]. Sound good?

## Template 3: Data Entry
**Subject:** Data entry — quick turnaround
**Body:** Hi, I can handle your data entry project. [X] records at $[rate]. Delivered in [time]. 100% accuracy guaranteed. Let me know!

## Template 4: Python Scripting
**Subject:** Custom Python script
**Body:** Hi, I can build a Python script that [function]. It will output [result]. Price: $[amount]. Includes documentation. Ready to start?

## Template 5: Web Scraping
**Subject:** Web scraping service
**Body:** Hi, I can extract [data type] from [website]. Output as CSV. Price: $[amount]. Done in [time]. Let me know!

## Template 6: Virtual Assistant
**Subject:** Virtual assistant — available now
**Body:** Hi, I'm available for [hours] hours/week. I handle email, research, data entry, scheduling. Rate: $[rate]/hr. Available to start immediately.

## Template 7: Content Writing
**Subject:** Content writer for your project
**Body:** Hi, I can write [X] articles/posts about [topic]. SEO-optimized, research-backed. $[price]/piece. Sample available on request.

## Template 8: Graphic Design
**Subject:** Design work — quick delivery
**Body:** Hi, I can create [design type] for your brand. Includes [revisions] revisions. Delivered in [time]. Portfolio: [link]. Price: $[amount].

## Template 9: Video Editing
**Subject:** Video editor — fast turnaround
**Body:** Hi, I can edit your raw footage into a polished [type] video. Includes cuts, transitions, music, color grading. $[rate]/hour or flat $[price].

## Template 10: Research
**Subject:** Research project — comprehensive report
**Body:** Hi, I can research [topic] and deliver a comprehensive report with sources, data, and insights. $[price]. Delivered in [time].
